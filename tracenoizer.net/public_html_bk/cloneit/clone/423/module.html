The console and parallelports are examples of char devices. Device drivers in Linux are known as modules and can be loaded dynamicallyinto the kernel using the insmod command. A single module can be compiled alone, and also can be linked to thekernel (here, care has to be taken on the type of driver). init_module loads the relocated module image into kernel space and runsthe module's init function. The module image begins with a module structure and is followed by codeand data as appropriate.
