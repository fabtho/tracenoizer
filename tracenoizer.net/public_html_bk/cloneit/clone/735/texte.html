Image Maps Eine Image Map ist ein Bild das "aktive Regionen" besitzt. Autoren sollten Client-side Image Maps (mit "usemap") benutzen, anstatt von Server-side Image Maps (mit "ismap"), weil letztere ein spezielles Eingabegeraet erfordern. dem Internet Explorer von Microsoft oder dem Netscape Navigator werden ALT Texte als sogenanntes ToolTip ueber dem Bild dargestellt, wenn man den Mauscursor darueber haelt. ALT Texte dienen als Alternative zum Bild weshalb sie auch nur dann dargestellt werden sollten, wenn man das Bild nicht sieht. Einige Autoren hat das verwirrt, und sie benutzen den ALT Text fuer andere Informationen als einen Bildersatz (Beispiele dazu im Horrorkabinett).
