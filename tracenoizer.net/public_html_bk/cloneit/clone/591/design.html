Using the principles developed in this research project, the prototype of the STG was assembled in less than half the time required for the assembly of the existing machine, which is going to be replaced by the STG.

Clay NACA TN 201 Sep 1924 A description is given of the methods used in design of Micarta propellers. The most direct method for working out the design of a Micarta propeller is to start with the diameter and blade angles of a wooden propeller suited for a particular installation and then to apply one of the plan forms suitable for Micarta propellers. This allows one to obtain the corresponding blade widths and to then use these angles and blade widths for an aerodynamic analysis.
