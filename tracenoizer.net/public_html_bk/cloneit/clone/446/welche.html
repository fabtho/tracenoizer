Viele Discount-Provider wie CompuServe oder T-Online erlauben ihren Mitgliedern keinen solchen Zugriff. Die meist verwendete Programmiersprache fuer CGI-Scripts ist Perl. Fragen Sie jedoch vor dem Programmieren eines eigenen CGI-Scripts Ihren Provider, welche Art von Programmen auf dem Server ausfuehrbar sind. Wenn Sie in Perl programmieren wollen, fragen Sie den Provider, ob der Server ueber einen Perl-Interpreter verfuegt, und wo genau der Interpreter auf dem Server liegt. .
