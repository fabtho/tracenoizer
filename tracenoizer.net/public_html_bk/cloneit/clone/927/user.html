Linux is different from other operating systems in how it separates user space(places were you can write data, save files to, and configure to your preferances) from system space ( places were files important to proper computer fuctioning reside). This makes linux very robust and less vulnerable to virus and arbitrary code execution attacks on the system. The multi user architecture keeps your data safe from the eyes of other users who might have access to the machine.
