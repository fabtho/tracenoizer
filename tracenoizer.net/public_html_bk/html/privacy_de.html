Persönliche Daten
Das TraceNoizer-Team ist verpflichtet die Privatsphäre der Benutzer der TraceNoizer website zu schützen. Jede (persönlich indentifizierbare) Information, welche von TraceNoizer durch die TraceNoizer website gesammelt wurde, kann verwendet werden um die Services von TraceNoizer auf Ihre spezifischen Interessen zuzuschneiden.

Ihre persönlich identifizierbare Information wird NICHT and Dritte weitergegeben oder verkauft. Als BenutzerIn der TraceNoizer website stimmen sie der Art und Weise zu, wie TraceNoizer diese Daten prozessiert.

Network Traffic
Das TraceNoizer Team überwacht den Network Traffic, um unbewilligte Versuche von Datenupload oder -download zu identifizieren. TraceNoizer sammelt Daten über den Network Traffic um die Leistung der website zu messen und um generelle statistische Information zu gewinnen über die Benutzung der TraceNoizer website.

Speicherung der gesammelten Daten
Die gesammelten Daten werden in einer sicheren Datenbank auf TraceNoizer's eigenen Servern gelagert, welche in einer sicheren Umgebung gehostet werden. Die gesammelten Daten können nur zu den Zwecken eingesehen werden, die in diesem Privacy Statement spezifiziert wurden.
