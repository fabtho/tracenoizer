1985 A great benefit of using Cold Fusion for Web-database connectivity is that developers can produce or modify Cold Fusion template files with any simple text editor; no special development software is necessary. The similarity of CFML to HTML is also a benefit for developers. On the other hand, one of the limitations of Cold Fusion is that template files cannot pull images from the database directly, although they can pull textual references to image locations. This does not limit what can be output to the Web but does mean that the developer, not the database, must handle the logistics of naming and storing image files. Another potential drawback for performance is that both CFML and the embedded SQL statements are interpreted rather than compiled.
