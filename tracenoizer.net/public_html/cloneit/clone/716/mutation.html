We are studying one particular aspect of germline mutation rates, namely the possible rate differences between sexes. If errors during DNA replication are the main source of germline mutations, we should expect the mutation rate to be closely correlated to the number of germline cell divisions. Since, in many organisms, the number of sperm greatly exceeds the number of egg cells produced, we might accordingly expect the mutation rate to be male-biased.
