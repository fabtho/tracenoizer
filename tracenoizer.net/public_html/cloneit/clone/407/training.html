NBCC Miramichi - Multimedia Learning Technology Learning Technologies Programs Multimedia Learning Technology Program Description The use of multimedia features is now the standard in the development of a wide range of software products, from information kiosks to complex training simulators. This 80-week program is designed to produce graduates who are entrepreneurs in the development of interactive educational software that is based on multimedia learning design. Students learn how to combine digital sound, video, animation, graphics and other media into interactive educational experiences.
