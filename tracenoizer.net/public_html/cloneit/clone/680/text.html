. CHAPTER 12 INTRODUCTION TO THE VI SCREEN EDITOR CHAPTER 12 - INTRODUCTION TO THE VI SCREEN EDITOR What is vi Vi is a screen editor. This means that you can see part of the file in a window on the screen, and editing operations can be controlled by moving a cursor around the text on screen. Vi works in a different way from the editing functions of modern word processors. It's effective use requires a considerable amount of expertise on the part of the user.
Abstract should be typed on a separate page and should not exceed 100 words. This is an essential part of the paper; lack of an abstract may delay publication. Tables should be typed on a separate sheet of paper and numbered consecutively with respect to citation in the text. They should be self-explanatory and should supplement, not duplicate, the text. Figures Type legends on a separate page at end of manuscript.
Authors should also supply a shortened version of the title suitable for the running head, not exceeding 50 character spaces. Each article should be summarized in an abstract of not more that 100 words. Avoid abbreviations, diagrams, and reference to the text. Good quality color prints should be provided in their final size. Figures needing reduction or enlargement will be charged an additional 25 percent.
A few methods are so well-known (chi-square test, for example), that is no longer necessary to cite an authority. ), and dates should follow the continental system (17 January 1985 rather than January 17, 1985). The text summarizes data found in the tables and figures but never restates it, number by number. Tables and figures must be mentioned or referred to in the written text, to direct the reader's attention to the appropriate figure at the appropriate time. Please give careful attention to the logical arrangement of each table so that it can be interpreted easily.
