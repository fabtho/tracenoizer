. An Instruction Scheduling Library for SUIF - Chen, Young (ResearchIndex) A. The library builds on existing control-flow graph, data flow analysis, and memory disambiguation infrastructure from the SUIF system with machine SUIF extensions. It provides support for alias analysis, dependency analysis, modeling machine latencies, and modeling resource requirements. It also supports top-down and bottom-up instruction scheduling, as well as adding instructions to scheduled blocks.
