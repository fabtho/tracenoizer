RAMP is a comprehensive framework within which risks can be managed effectively and financial values placed upon them. It aims to achieve as much certainty as possible about a long term and uncertain future. The process facilitates risk mitigation and provides a system for the control of the remaining risks. It pays special attention to disaster scenarios, even if the probability of occurrence is low. .
