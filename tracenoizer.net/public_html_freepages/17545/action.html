Terminating a Process When you use the Process Control action to terminate a process, enter the name of the process you wish to terminate, exactly as it appears in the Process List in the Task Manager. . For example, if you have a mission critical program, you can create a Process Monitor that watches to make sure the program is running, and link it to a Process Control action to automatically restart the program if it terminates. The Process Control action can also be used as a generic way to extend the functionality of Appmon, since it can be used to execute any arbitrary command line. You can specify a limit to restrict the number of times Appmon will attempt to start or stop the process in a given time period.
