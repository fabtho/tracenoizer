In selecting an item, the price is shown with a collection of money. This activity is identified as kinesthetic learning. The groups are given a combination of coins and dollars and instructed to find and write the total. This consists of students working on two Student Pages from the student edition. The teacher guide provides a paragraph or two of guidance for using each of these pages.
