Each software package is modular where extended features are supported through plug-in software modules. Each package also includes an optional data acquisition module allowing the collection of raw data directly from instrumentation. PRAP (Piezoelectric Resonance Analysis Program) This software package allows the determination of complex elastic, piezoelectric, and dielectric properties of piezoelectric resonators. Included with analysis is the computation of components of Van Dyke's circuit model and those of a complex circuit model. An optional data acquisition module integrates the collection of resonance spectra from an impedance analyser.
