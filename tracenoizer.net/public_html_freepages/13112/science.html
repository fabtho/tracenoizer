The Science Data User's SFDU Workbench The Science Data User's SFDU Workbench The Science Data User's Workbench was developed at Goddard Space Flight Center (GSFC) to provide users of scientific data with an easy to use tool for creating and using data conforming to the Consultative Committee for Space Data Systems (CCSDS) Standard Formatted Data Unit (SFDU) concept (sometimes known as the Standard For Data Understanding). The SFDU Workbench is being developed in C on a Sun SPARC IPX running Sun OS Version 4. For more complete information, please consult the User's Guide.
