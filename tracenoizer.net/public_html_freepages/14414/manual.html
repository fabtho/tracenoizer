Psion to Mac Text Qualifer should be None, and Label Separator should be Tab, as before. Entry separator should be set to Other, and then you have to name a character to use. It can be any character that hasn't been used in the text (you aren't restricted to characters that can be typed directly - see the Psion manual about special characters - but these special characters may not show up properly on the Mac - it will depend on the font used). Now replace all CarriageReturn LineFeeds by a string not used anywhere - qqq should be safe - and then replace the character used as record separator by CarriageReturn. Further checks suggest that the label separator just has to be deleted - equivalently, do a Replace All, replacing it with nothing.
