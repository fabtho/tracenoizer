Data in the database includes the available numbers and condition of school infrastructure, supporting services, equipment and resources. The SRN database is a Geographic Information System (GIS) containing point data on school locations as well as a range of data fields on infrastructure components linked to it. Work to be done by Praxis includes Upgrading and customisation of the existing SRN database Linking of SRN data with additional data fields from other databases Development of a system that includes procedures, processes and software applications with which the Department of Education will be able to continuously update and maintain the SRN database Development of an easy to use planning and research tool for the provincial Departments of Education Capacity building and training of national and provincial departmental officials in the optimal utilisation of the system for planning and research. The work will be done in Microsoft Visual Basic, Access and SQL Server, with connections to a GIS system and possibly web access. Please contact Tracy Millar at Praxis for more information.
