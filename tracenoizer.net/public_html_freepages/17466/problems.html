FuturePIM FuturePIM multicast routing software is a portable implementation of PIM-SM and PIM-DM providing multicast routing and multicast forwarding capability to a router. FuturePIM is designed to provide scalable inter-domain multicast routing across the Internet and avoids the performance problems of earlier multicast routing protocols. FuturePIM is independent of the underlying unicast routing protocol using only the unicast routing information. FuturePIM provides comprehensive management support based on a MIB.
