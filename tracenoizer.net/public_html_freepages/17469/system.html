If your device or application does not require the support provided by a RTOS, the RomPager Web server can still be easily integrated with your embedded application. This means that your products can send asynchronous status reports using email, the universal Internet application. For example, a network printer using RomMailer could send an email to a system administrator when consumables such as toner or paper needed to be replenished or to the printer vendor's service organization when preventive maintenance needs to be scheduled. The RomPOP Embedded Email Toolkit is a Post Office Protocol version 3 (POP3) client that allows your embedded devices to receive email from any POP3 server. Your embedded products can receive asynchronous status reports and information using the universal Internet application, email.
