There has been scattered work on the console and TTY layer going on for some time. This project will try to pull it all together into a coherent architecture from which we can generate clean patches for Linus and the core kernel team. The various subsystems are intertwined with the console system. Plus with the new design it is possible to use framebuffer devices without a VT. The console system input system is based solely on the input api.
It also contains a brief list of links to other, related resources about visual language research. The papers in the bibliography were all contributed by their original authors. The original classification system has since been expanded to handle some other subareas of VL research. To navigate through the bibliography, you may click at any particular section in the table of contents to view its entries, or you may manually scroll down through all the sections. All headings in the table of contents are used to classify papers, not just the lowest levels.
