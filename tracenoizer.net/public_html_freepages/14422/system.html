The Falcon Monitoring and Steering System The Falcon Monitoring and Steering System Falcon was a set of tools that collectively support on-line program monitoring and steering of parallel and distributed applications. Easy-to-use default monitoring and graphical displays for all applications. Graphical user interface for instrumentation and run-time manipulation FALCON Release Information The Bigger Picture Falcon was part of a larger project to create Distributed Laboratories.
