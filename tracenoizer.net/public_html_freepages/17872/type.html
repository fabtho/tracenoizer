Choose the size of your type from the Size drop-down menu in the right hand column. .The default type size (basefont) is 3; it roughly corresponds to 10 point type. You will want to use larger type sizes for headlines and smaller type sizes for your navigation bar and address footer. There are several ways to change the size of your type in FrontPage 98.

Subtle progressive icons on primary navigation -- useful for color blind users -- are a visual cue and a good substitute for the color shift between primary sections that other users see. Use of stylesheets (CSS) to embolden and color text links differently from surrounding text. The site uses sans-serif type, which has been proven to be more readable onscreen. Large, distinct header style makes it easy to differentiate headers from other content and to skim through pages easily. Colors Color is used to differentiate sections both in the primary navigation at the top of each page, and in the secondary navigation column on the left of each page.
