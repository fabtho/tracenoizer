- data -

D C Mason et al from the NERC Environmental Systems Science Centre has an approved Special Topic proposal in this area that will use ERS-1 and ERS-2 SAR data and CASI data to define beach profiles and to combine these data with land and hydrographic information in a single bathymetry and topography data set for the LOIS community. . This will be achieved by setting up five Data Centres that will operate to agreed protocols in serving the LOIS community, providing repositories for the large quantities of data that will be produced by the component programmes (particularly the large data sets from the airborne remote sensing programme, moored continuous monitoring instrumentation and gauging stations). At the same time the Data Centres will make quality-assured data available for analysis, graphic presentations, interpretation and modelling through linking databases to GIS. LOIS will make a major investment in data gathering and collation.
