Institutional Master File The IMF data sets are rectangular flat files containing numeric and character data. The size of each file varies according to the reporting year. The unit of analysis for all records within each file is the institution (the individual public or private school, BOCES, or District. Each record begins with a header section identifying the institution type, its name and location, and the name of its chief officer. Each record also includes data on the facility's number and type of staff, enrollment statistics, current programs, and educational resources.
