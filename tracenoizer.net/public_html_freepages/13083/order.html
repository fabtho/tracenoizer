Even though your laptop is probably using Advanced Power Management (APM) or the Advanced Configuration and Power Interface (ACPI), your USB device will still cause the system to use more power. USB devices require a certain amount of bus activity in order to maintain communication with the USB device. That activity prevents the CPU from switching to the C3 (or clock stopped) power-down state. Since the CPU normally can spend a lot of time in the C3 state, this can save a great deal of power.
