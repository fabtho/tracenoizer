If an agent observes a sign, there is a chance the agent may interpret a meaning different from that of the agent who created the sign. And there is a chance that the observer may see an object that is accidentally created, and looks like a sign, and interpret it as having some meaning. This doesn't change the fact that the sign itself has a meaning for the agent. So, if I find a four leaf clover, and consider it a sign of good luck, the fact that it is accidental does not change the fact that I consider it a sign of good luck.
