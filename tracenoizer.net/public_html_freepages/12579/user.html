. WET is a program for phylogenetic reconstruction using sequence data. The main goal in the development of WET was to make a really user friendly program able to interact with other phylogenetic packages. It also includes a user editable matrix for weights to different kinds of nucleotide substitutions. Installing the program Just copy it to the desired directory and execute it.
