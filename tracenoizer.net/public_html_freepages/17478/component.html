The command has a large number of options and an atom-selection, see section Atom Selection. The atom selection allows the user to select any portion of the system for the calculation. For example, if one is interested in the calculation of the binding energy of a complex, the atom selection can be used to select each component and then all the atoms for separate electrostatic calculations. Option Purpose SOLVENT Specifies the dielectric constant of the solvent in units of the vacuum dielectric. Thus, the vacuum dielectric would be 1, and the dielectric constant of water is around 78.
