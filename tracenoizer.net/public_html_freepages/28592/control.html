An invariant is an expression which must evaluate to true whenever the entity being described is in a stable state. An invariant expression must evaluate to a boolean. An object is defined as being in a stable state when either (a) no threads of control are within the object or operating upon the object, (b) a single thread of control is about to enter a method of the object, or (c) a single thread of control is about to exit a method of the object.
