The Cooperative Peer Response System (CPR) is a discussion forum collaboration tool developed at the ARIES Laboratory (Laboratory for Advanced Research in Intelligent Educational Systems) in the Computer Science Department at the University of Saskatchewan. CPR offers an easy-to-use interface and a powerful, flexible administration system. It also traps a rich quantity of user activity data. Future work on CPR includes developing tools that create user profiles from this user activity data. Finally, CPR is built using a multi-tier system consisting of a database, application server and web browser client.
