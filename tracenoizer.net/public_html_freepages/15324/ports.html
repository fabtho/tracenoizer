webmin is not listening on a port defined in some rfc, the port is specified during the installation process. I usually use port 900 (cause Samba is on port 901 by default. if there's no one to connect to your machine on the high ports it's safe to close them. If the system is only serving mail and perhaps dns, you need only two or three ports. DNS might need a unpriviliged port for querying a dns root server.
