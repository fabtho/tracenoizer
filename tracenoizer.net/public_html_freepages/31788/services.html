Monitoring communication lines is strictly forbidden. You must not take any action which might damage hardware, software or data stored on any computer. No network services (aka servers) are to be deployed except those authorized by the computation center. By "servers" we mean HTTP servers, mail servers, file servers etc. It is forbidden to install a software that allows to take control remotely of a computer located inside the univerity's facitlies (like PCanywhere servers), unless it has been approved in advance by the computation center.
