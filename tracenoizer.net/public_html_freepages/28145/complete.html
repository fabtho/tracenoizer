I have taken a stab at defining what I feel would be a good "Director's Cut" of this movie which blends the best of all the available versions. Maybe Amy Heckerling will one day agree to release something similar to this as an offical "Director's Cut" as has been done for Blade Runner and other films. My approach to creating this director's cut was to start with the most complete version, the broadcast TV version, and edit from there. As a result, if a scene is the same in all the versions I usually selected the broadcast TV version as the source. Interestingly, this edit comes closest of all to following the novel and screenplay.
