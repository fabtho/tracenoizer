The Studio contains communication tools (news group in www and real time Java chat with whiteboard) and applications for creating mind maps (Java-applet) and WWW- based multimedia presentations (WWW-editor). The Meeting Room contains same communication tools as the Studio plus applications for cooperative work and for making new learning materials. The Gallery is a space of exhibition of presentations made by the students. The report will be published in the end of January 1998. In future the learning environment and the materials can be offered for use in other school districts in Finland and abroad.
