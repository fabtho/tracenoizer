We felt that we wanted to support EML and to raise awareness about it. FLe3 is a specialized application that has been designed to help study groups interested in collaborative learning to engage in research-type learning processes. The system makes use of the progressive enquiry process of knowledge building. The starting point is to create a context for the study project in order to anchor the chosen issues to the central principles of the knowledge domain in question. This helps to gives students an understanding of why the issues in question are worth studying.
The new release corrects some bugs and problems of the earlier version. The new version contains also several improvements to the user interface. For instance, new version contains several key features helping Course Administrators in their instructional design and organizing of fruitful knowledge building processes. Please, note that if you already have a working copy of the Fle2 server, you should not reinstall the whole Fle2 package but upgrade your server. Also the Fle2 server, hosted by the UIAH Media Lab has been updated with the new server software.
