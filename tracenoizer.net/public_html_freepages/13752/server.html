2 Shell API The simplest use of Festival (though not the most powerful) is simply using it to directly render text files as speech. festival --tts This is easy to use but you will need to wait for Festival to start up and initialise its databases before it starts to render the text as speech. A socket based server mechanism is provided in Festival which will allow a single server process to start up once and be used efficiently by multiple client programs. For many application this will be the preferred interface method. Other text modes too are available through the command line by using auto-text-mode-alist.
