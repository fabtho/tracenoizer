USC chimera detection software server Welcome to the USC chimera detection software server The USC chimera detection server uses the chidetect and mglobalCHI software packages, developed by George Komatsoulis and Michael Waterman to help determine if an environmentally derived 16S rRNA sequence is chimeric. A UNIX version of this software is also available for download from the software section of our server. Users unfamiliar with the system are encouraged to study the online user manual and the man pages for chidetect and mglobalCHI.
