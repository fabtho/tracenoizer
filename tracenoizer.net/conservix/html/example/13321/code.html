The CueCat The CueCat is a barcode scanner passed out with subscriptions to Forbes magazine and the Radio Shack catalog. The idea is that consumers will install the CueCat software and use the scanner to scan in the bar code associated with certain advertisments. Once the bar code is scanned into the computer, the software will direct the users browser to the products web page. Essentially it is a way of giving the user of the CueCat immediate, up to date information on a product the user is interested in.
The free scanner can be picked up at any RadioShack, or may be ordered for mail delivery at the CueCat web site (the mail offer also includes a video convergence cable in the shipping charge). The motivation behind this free giveaway is for people to use these scanners to swipe ordinary product UPC's as well as special bar codes called cues that appear in the Radio Shack catalog and other print publications like Wired and Forbes. The scanner operates off your computer keyboard port, so this swiping action causes the CRQ software included with the unit to launch your web browser and go to the specific web page for the bar code that was swiped. The convergence cable does a similar thing for your TV.
