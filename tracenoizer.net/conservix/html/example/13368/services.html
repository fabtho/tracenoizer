System Accounts FreeBSD Handbook Prev Chapter 8 Users and Basic Account Management Next 8. 4 System Accounts System users are those used to run services such as DNS, mail, web servers, and so forth. The reason for this is security; if all services ran as the superuser, they could act without restriction. Examples of system users are daemon, operator, bind (for the Domain Name Service), and news.
