Getting EMacro

Download EMacro here. EMacro is now packaged for the most popular Operating Systems, explained in the download area. The following instructions help you install the tar.gz package.
If you run Microsoft Windows, you will need tar & gzip to extract the EMacro files. You can do this with Cygwin, from http://sources.redhat.com/cygwin/ or UWIN from http://www.research.att.com/sw/tools/uwin/. After installing it, run the bash shell or korn shell. Then run commands like:

  • cd /home/emacs
  • tar -xzf emacro*.tar.gz

There are also some graphical GUI archive tools:

  • 7-zip free GPL
  • FilZip no cost
  • Ultimate Zip no cost

If you run Mac OS (older than version 10), you can get tar and gzip software to extract EMacro from http://www.pure-mac.com/compen.html. You might also be able to use Stuffit Expander >= v6.0

Also visit http://my.gnus.org/GDG/basic/show_form to create your ~/.gnus for email and newsreading with Emacs.
